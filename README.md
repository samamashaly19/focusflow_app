##  Getting Started

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- Android Studio / VS Code

### Installation

```bash
# Clone the repository
git clone https://github.com/samamashaly19/focusflow_app.git

# Go to project directory
cd focusflow_app

# Install dependencies
flutter pub get

# Run the app
flutter run
